# PROJET-DATA-
#  Alternance Readiness Index Dashboard

##  Project Overview
This project aims to answer a simple but important question:

**"Is a student truly ready for an apprenticeship?"**

Recruiters often rely on subjective impressions such as motivation, seriousness, or potential.  
This project transforms these vague criteria into measurable data indicators.

---

##  Objective
To build an interactive dashboard that evaluates student readiness for apprenticeships using a data-driven scoring system.

---

##  Key KPIs

- Alternance Readiness Score (out of 100)
- Technical Skills Score
- Consistency Score
- Visibility Score (LinkedIn / GitHub)
- Progress Tracking Over Time

---

##  Technologies Used

- Python (Pandas, Matplotlib)
- SQL (data structuring)
- Power BI (interactive dashboard)

---

##  What This Project Demonstrates

✔ Ability to turn a business problem into a data solution  
✔ Data visualization and storytelling skills

---

##  Impact

This dashboard helps recruiters to:

- Objectify subjective evaluation criteria
- Support HR decision-making
- Identify candidate potential
- Track student progress
